# Онлайн-библиотека

Курсовой проект на тему "Разработка веб-интерфейса для онлайн-библиотеки".

Проект выполнен на базовом стеке фронтенд-технологий:

- HTML
- CSS
- JavaScript

## Описание

Веб-интерфейс позволяет просматривать каталог книг, искать книги по названию или автору, фильтровать список по жанру, добавлять книги в пользовательский список "Мои книги", а также открывать небольшую карточку с описанием книги и выставлять оценку.

## Основные функции

- отображение каталога книг;
- поиск по названию и автору;
- фильтрация по жанру;
- добавление книг в список "Мои книги";
- удаление книг из выбранного списка;
- просмотр описания книги в отдельной карточке;
- выставление оценки книге от 1 до 5;
- добавление новой книги через форму.

```text
WebApp/

```

## Запуск

Откройте файл `WebApp/index.html` в браузере.

Также можно запустить простой локальный сервер из папки `WebApp`:

```bash
node -e "const http=require('http'),fs=require('fs'),path=require('path');const root=process.cwd();const types={'.html':'text/html; charset=utf-8','.css':'text/css; charset=utf-8','.js':'text/javascript; charset=utf-8'};http.createServer((req,res)=>{let url=decodeURIComponent(req.url.split('?')[0]);if(url==='/')url='/index.html';const file=path.join(root,url);fs.readFile(file,(err,data)=>{if(err){res.writeHead(404);return res.end('Not found')}res.writeHead(200,{'Content-Type':types[path.extname(file)]||'text/plain; charset=utf-8'});res.end(data)})}).listen(5500)"
```

После запуска страница будет доступна по адресу:

```text
http://localhost:5500
```

## Назначение проекта

Проект предназначен для демонстрации базовых навыков разработки клиентского веб-интерфейса: семантической HTML-разметки, адаптивной CSS-верстки, работы с DOM, обработки событий и управления данными на стороне браузера.
